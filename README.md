# Mesh AI Network

## Overview
A Flask-based AI-powered mesh network system for intelligent communication between distributed nodes.

## Features
- Real-time node communication
- AI-based response handling
- Lightweight Flask server
- Mobile-access support

## Tech Stack
- Python
- Flask

## Setup Instructions

### 1. Clone repo
git clone <your-repo-link>

### 2. Install dependencies
pip install -r requirements.txt

### 3. Run app
python app.py

## Access
Local: (http://127.0.0.1:5001) 
Mobile: http://YOUR_IP:5001
